Map, Filter, Reduce

Higher Order Functions

Mapping means to apply a function to each and every element of an array

  • .map returns an array of the same length as the input
  • .map takes in a function body and applies it to each and every element of the input array
  • .map is the right tool for transforming arrays and keeping them the same length

Filtering means to filter an array based on a function that returns a boolean

  • .filter returns an array the same length or shorter than the input
  • .filter takes in a function body and uses the function body as the filter
  • .filter's function argument it takes in should return a boolean

Reducing means to apply a function to an array and return a single data type

  • .reduce returns any single data type (array, object, number, string, etc..)
  • .reduce takes in a function and applies it to all elements of the array in order to produce a single answer
